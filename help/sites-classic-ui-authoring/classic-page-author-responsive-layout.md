# Disposition réactive{#responsive-layout}

>[!CAUTION]
>
>Bien que le composant Conteneur de mises en page soit disponible dans l’IU classique, il n’est entièrement fonctionnel et pris en charge que dans l’interface utilisateur optimisée pour les écrans tactiles. Pour plus d’informations, veuillez consulter [Disposition réactive](/help/sites-authoring/responsive-layout.md) dans la documentation de création standard.
